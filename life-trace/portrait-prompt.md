# 历史人物贴纸生成 Prompt

将 `{{变量}}` 替换为已核对的人物资料；缺少的事迹或装饰变量直接删除，不编造内容。

```text
生成一张以中国历史人物“{{人物姓名}}”为主体的 1:1 剪纸拼贴风人物贴纸，用于人生足迹地图的人物卡片。

人物主体：画面中央为{{朝代}}时期的{{人物身份}}“{{人物姓名}}”半身像。外貌为{{外貌特征}}，身穿{{服饰描述}}，手持或佩戴{{代表性物品}}，气质{{人物气质}}。使用稍粗的黑色墨线，呈现宣纸剪贴、木刻版画、旧印刷颗粒和手工贴纸质感；人物完整清晰，不写实。

文字布局：
- 右上方在深红或朱砂色撕纸上竖排姓名“{{人物姓名}}”，中文必须准确清晰。
- 姓名附近放旧纸黄色竖向称号“{{人物称号}}”。
- 左侧放 3—4 组旧宣纸竖签：“{{著名词语1}}”“{{著名词语2}}”“{{著名词语3}}”“{{著名词语4}}”；搭配少量墨环、朱砂圆点或破损纸边。
- 底部旧宣纸横条写：“{{人物简介}}”，控制在 25—45 个汉字，字号清晰可读。
- 加入 1—2 枚旧印泥质感的朱砂印章：“{{人物字号或精神关键词}}”“{{朝代简称或代表性品格}}”。

风格与构图：黑色、深红、米色、铜金和旧纸黄为主；可少量使用竹简、古籍残页、半调网点、祥云、传统纹样和毛笔笔刷。装饰不遮挡面部，姓名在人物右上方，事迹在左侧，简介在底部，所有元素形成一个完整贴纸轮廓。主体边缘加略粗的米白色贴纸描边，方便地图页面展示与印刷。

背景必须为真正透明背景，只保留人物与贴纸装饰。边缘清晰，无阴影底板、无场景背景。

禁止：不要白色棋盘格背景，不要把透明背景画成灰白方格；不要错字、乱码、伪汉字、重复文字或过小文字；不要写实照片、电影海报、3D 建模、西方油画、霓虹灯、高饱和渐变、现代商业海报、复杂真实场景、模糊边缘或水印。
```

生成后必须人工核对姓名、称号、事迹、简介与印章文字，并确认 PNG 具有透明通道；文字错误或棋盘格被画进图片时不得直接交付。
